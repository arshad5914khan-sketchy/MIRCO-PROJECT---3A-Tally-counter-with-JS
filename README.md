# MIRCO-PROJECT---3A-Tally-counter-with-JS
🔢 Micro Project 3 A| A simple Tally Counter built with HTML, CSS &amp; JavaScript. Increment, decrement, and reset a counter with clean UI.

 🔢 Tally Counter | Micro Project - 3 A

A simple and interactive Tally Counter built using HTML, CSS, and JavaScript as part of my self-directed Full Stack learning journey.

About the Project
This is the first part of my **Micro Project Series - 3**, where I build small, focused projects to strengthen my fundamentals in HTML, CSS, and JavaScript before moving on to more complex applications.

The Tally Counter lets users:
-  Increase the count
-  Decrease the count
-  Reset the count back to zero

 Tech Stack
- HTML – Structure of the counter and buttons
- CSS – Styling for a clean, minimal UI
- JavaScript – DOM manipulation and event handling for counter logic

 What I Learned
- DOM selection and manipulation using `document.querySelector`
- Handling click events with `addEventListener`
- Updating UI dynamically without reloading the page
- Structuring a small project with separate HTML, CSS, and JS files

 How to Run
1. Clone this repository
2. Open `index.html` in your browser
3. Click Increase, Decrease, or Reset to interact with the counter
